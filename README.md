# 🤖 Bot Discord em Python

Bot desenvolvido com Python utilizando a biblioteca discord.py, com foco em automação e gerenciamento de servidores.

---

## ⚙️ Funcionalidades

- 📩 Sistema de tickets  
- ⚙️ Comandos personalizados  
- 🔄 Automações  
- 🌐 Integração com APIs  
- 🛠️ Organização em cogs  

---

## 🛠️ Tecnologias utilizadas

- Python  
- discord.py  
- Git  

---

## 🚀 Como executar o projeto

```bash
git clone https://github.com/eusants01/bot-discord
cd bot-discord
pip install -r requirements.txt
python main.py
